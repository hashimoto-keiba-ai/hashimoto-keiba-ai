# 函館OS v1.0

## 目的

函館競馬場の過去データを橋本競馬AIの競馬場学習DBへ接続し、函館開催用の初期補正ルールとして利用します。

## コース傾向

- 洋芝適性を重視する
- 小回り適性とコーナー加速を評価する
- 滞在競馬で状態を上げる馬を加点する
- 開催後半は馬場悪化と内外バイアスを確認する

## 補正ルール

- 洋芝実績あり: 加点
- 函館・札幌好走歴あり: 加点
- 先行力と器用さあり: 加点
- 時計勝負専用タイプ: 軽い減点
- 長距離輸送後の初戦: 状態確認を優先

## 学習DB接続

このMarkdownは `racecourseLearningDatabase` の `learningSignals` として読み取り、函館競馬場の初期OSデータとして扱います。

## 次回改善点

- 函館開催の結果検証データを追加する
- 洋芝適性と人気ゾーンの相関を集計する
- 高配当パターンが出たレースの脚質と枠順を検証する
