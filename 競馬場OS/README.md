# 競馬場OS

## 用途

主要8場のコース傾向、馬場バイアス、枠順、脚質、距離別ロジックを横断管理する共通フォルダです。

各競馬場フォルダに蓄積した現場データを、橋本競馬AIの共通判断ルールとして再利用するために使用します。

## 対象競馬場

- `中山競馬場/`
- `東京競馬場/`
- `京都競馬場/`
- `阪神競馬場/`
- `中京競馬場/`
- `新潟競馬場/`
- `福島競馬場/`
- `小倉競馬場/`

## 管理する情報

- 競馬場別の芝・ダート、距離、内外回りの基本特性
- 開催時期、馬場状態、天候によるバイアス
- 枠順、脚質、ペース、上がり性能の効き方
- AI指数、危険人気馬、神レース判定へ反映する共通補正ルール
