# 橋本競馬AI

橋本競馬AIは、AI指数ランキング・神レースランキング・万馬券候補ランキング・危険人気馬・WIN5生成・競馬場別AI・騎手ランキングを管理する競馬予想プロジェクトです。

## ダッシュボード

トップ画面は `index.html` から開けます。

- AI指数ランキング
- 神レースランキング
- 万馬券候補ランキング
- 危険人気馬
- WIN5生成
- 競馬場別AI
- 騎手ランキング

## 整理後のフォルダ構成

### 競馬場別AI

競馬場別の分析は、以下の8フォルダに統一して管理します。

- `中山/`: 中山開催のコース傾向、馬場、脚質、枠順、レース総括
- `東京/`: 東京開催のコース傾向、直線適性、展開、過去結果
- `京都/`: 京都開催のコース条件、馬場、脚質、枠順、過去結果
- `阪神/`: 阪神開催の坂適性、距離別成績、馬場、脚質傾向
- `中京/`: 中京開催の左回り、急坂、距離別傾向、馬場、脚質傾向
- `新潟/`: 新潟開催の直線競馬、外回り、長い直線、馬場、脚質傾向
- `福島/`: 福島開催の小回り適性、先行力、距離別傾向、穴馬条件
- `小倉/`: 小倉開催の小回り、平坦コース、開催時期別の馬場差

札幌・函館は今回の主要8場とは分け、`その他競馬場/` 配下に保管します。

### 研究・運用領域

- `AI研究所/`: AI指数エンジン、危険人気馬AI、神穴AI、自己進化AIなどの研究メモ
- `WIN5/`: WIN5 AI、対象レース、買い目、結果検証
- `万馬券DB/`: 万馬券のレース記録、穴馬条件、高配当パターン
- `学習ログ/`: AI学習の実行記録、精度、失敗例、改善点
- `結果検証/`: 予想結果、着順、回収率、的中率、反省点
- `data/`: レース結果、出走表、オッズ、馬場情報、加工前後データ
- `docs/`: 設計メモ、運用ルール、分析方針、手順書
- `src/`: ダッシュボードのスタイルや今後の分析プログラム
