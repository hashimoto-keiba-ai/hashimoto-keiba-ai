# 橋本競馬AI

橋本競馬AIは、AI指数ランキング・神レースランキング・万馬券候補ランキング・危険人気馬・WIN5生成・競馬場別AI・騎手ランキングを管理する競馬予想プロジェクトです。

## ダッシュボード

トップ画面は `index.html` から開けます。

- AI指数ランキング
- 神レースランキング
- 万馬券候補ランキング
- 危険人気馬
- WIN5生成
- 競馬場別AI
- 騎手ランキング

## フォルダ構成

橋本競馬AIの管理領域を、競馬場別フォルダと共通フォルダに分けて整理しています。

### 競馬場別フォルダ

主要8場の分析は、以下の8フォルダに統一して管理します。

- `中山競馬場/`: 中山開催の急坂、内回り・外回り、馬場、脚質、枠順、レース総括
- `東京競馬場/`: 東京開催の長い直線、直線適性、展開、馬場、過去結果
- `京都競馬場/`: 京都開催のコース条件、馬場、脚質、枠順、過去結果
- `阪神競馬場/`: 阪神開催の坂適性、距離別成績、馬場、脚質傾向
- `中京競馬場/`: 中京開催の左回り、急坂、距離別傾向、馬場、脚質傾向
- `新潟競馬場/`: 新潟開催の直線競馬、外回り、長い直線、馬場、脚質傾向
- `福島競馬場/`: 福島開催の小回り適性、先行力、距離別傾向、穴馬条件
- `小倉競馬場/`: 小倉開催の小回り、平坦コース、開催時期別の馬場差

札幌・函館は今回の主要8場とは分け、`その他競馬場/` 配下に保管します。

### 共通フォルダ

競馬場を横断して使う研究・DB・運用領域は、以下の共通フォルダに集約します。

- `AI研究所/`: AI指数エンジン、危険人気馬AI、神穴AI、自己進化AIなどの研究メモ
- `WIN5/`: WIN5 AI、対象レース、買い目、結果検証
- `万馬券DB/`: 万馬券のレース記録、穴馬条件、高配当パターン
- `危険人気馬DB/`: 過剰人気、展開不利、馬場不一致などの危険人気馬データ
- `神レースDB/`: 期待値、軸信頼度、荒れ指数が高い勝負レースの記録
- `騎手データベース/`: 騎手別の競馬場適性、距離適性、脚質別成績、人気別信頼度
- `調教師データベース/`: 調教師別の仕上げ傾向、ローテーション、競馬場別成績
- `競馬場OS/`: 主要8場のコース傾向や馬場バイアスを横断管理する共通補正ルール

### 補助フォルダ

- `学習ログ/`: AI学習の実行記録、精度、失敗例、改善点
- `結果検証/`: 予想結果、着順、回収率、的中率、反省点
- `data/`: レース結果、出走表、オッズ、馬場情報、加工前後データ
- `docs/`: 設計メモ、運用ルール、分析方針、手順書
- `src/`: ダッシュボードのスタイルや今後の分析プログラム
