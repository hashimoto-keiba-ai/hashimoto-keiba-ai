# 橋本競馬AIロードマップ（AI研究所モード）

作成日: 2026-06-02

## Phase1: 完成済み

- 総合司令塔UIをトップ画面へ追加し、神レース、本日の勝負レース、AI指数TOP10、神穴TOP10、危険人気馬TOP10、WIN5候補、保存ログ検索、回収率/ROI、AI自己進化ログを横断監視できるようにする。
- 今日のレース一覧、実データ入力フォーム、AI指数自動入力、三連単自動生成、WIN5自動生成を同じダッシュボード内で運用する。
- 結果検証フォーム、最新結果検証ログ、OS自動アップデート管理、AI自己進化ループを接続し、検証から改善へつながる保存導線を作る。
- localStorage保存、JSONバックアップ、復元前バックアップ、安全モードを使って既存機能を壊さず運用できる状態にする。

## Phase2: 実装中

- GitHub Contents APIへの実保存/読込を実装し、`data/*.json` とブラウザ保存の同期を安全に行う。
- GitHub保存前自動バックアップ、SHA衝突検知、保存失敗時の復旧メッセージを強化する。
- 結果検証ログ、ROIログ、自己進化ログを横断集計し、AI指数、展開AI、神穴AI、危険人気馬AI、WIN5 AIへの補正案をより詳細化する。
- OSアップデートの採用/保留/削除ルールを、競馬場別・距離別・馬場別に再利用しやすい形へ整理する。

## Phase3: 将来実装

- 出馬表、オッズ、馬場状態、払戻結果などの実データ取得を自動化する。
- 機械学習モデルでAI指数、神穴指数、危険人気馬指数、WIN5候補を再計算し、特徴量重要度とモデル評価を表示する。
- スマホ通知で神レース昇格、危険人気馬急上昇、WIN5候補変更、ROI低下を知らせる。
- 競馬場別Markdownログの自動生成、マルチユーザー権限、レビュー承認フローを追加する。
- 予想、買い目、検証、改善、次走反映までのAI自己進化サイクルを半自動化する。
