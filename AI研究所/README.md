# AI研究所

## 用途

予想モデル、特徴量、評価指標、改善案、仮説検証メモを整理するフォルダです。

橋本競馬AIの分析ロジックや学習方針を研究する中心フォルダとして使用します。

## 構成

- `AI指数/`: 能力、適性、馬場、展開、騎手・厩舎補正を統合した指数研究領域
- `展開AI/`: ペース、隊列、位置取り、脚質有利不利を分析する研究領域
- `神穴AI/`: 人気薄の好走条件と期待値を抽出する研究領域
- `危険人気馬AI/`: 過剰人気や凡走リスクを検出する研究領域
- `AI指数エンジン/`: 基礎能力、コース適性、馬場補正、展開補正を統合する中心指数
- `自己進化AI/`: 予想結果から重み付けや判定条件を改善する研究領域
- `橋本競馬AI_完全設計図.md`: 全体設計、競馬場別フォルダ、共通DB、競馬場OSの管理方針
