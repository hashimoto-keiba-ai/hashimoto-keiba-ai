# Nakayama OS Updates

Racecourse-specific operating system updates for Nakayama.

## OS Updates Manager

Use the Phase8-5 OS Updates Manager to store Nakayama OS rules in localStorage and connect them back to this racecourse database.

Manager fields:
- Racecourse
- Distance
- Surface
- Condition
- OS Rule Title
- OS Rule Content
- Verification Race
- Adoption Date
- Importance: Critical, High, Medium, Low

Search and filter:
- Search by rule title, content, distance, surface, condition, verification race, adoption date, or importance.
- Filter by importance level.

Manager screen: [Phase8-5 OS Updates Manager](../../../phase8-racecourse-management.html)

## Stores

- Rule changes
- Weight adjustments
- Exclusion conditions
- Next validation points
