# CHANGELOG

## [1.0.0] - 2026-06-02

### Added

- 黒×金デザインの橋本競馬AI総合ダッシュボードを整備しました。
- PC/iPad/スマホに対応したレスポンシブUI、iPad向け2カラム表示、スマホ向け1カラム表示、下部固定ナビゲーションを追加しました。
- 今日のレース一覧、神レースTOP10、AI指数ランキング、神穴ランキング、危険人気馬ランキングをトップ画面から確認できる構成にしました。
- 実データ入力フォームで、レース条件、出走馬、人気、想定オッズ、脚質、AI指数、神穴指数、危険人気馬指数を管理できるようにしました。
- CSV取込機能により、ブラウザ上で運用データを取り込めるようにしました。
- AI指数自動入力、危険人気馬自動判定、神穴候補管理の運用導線を追加しました。
- 三連単自動生成機能を追加し、本線・攻撃型・万馬券型の買い目管理に対応しました。
- WIN5自動生成機能を追加し、A固定、B本線、Cゾーン、D爆発候補を整理できるようにしました。
- ROI/回収率ダッシュボードを追加し、投資額、払戻額、損益、的中率、券種別・競馬場別・月別集計、資金配分AIを管理できるようにしました。
- localStorage保存に対応し、レース入力、出走馬、予想、結果、OSアップデート、WIN5、ROI、買い目、バックアップデータをブラウザ内に保存できるようにしました。
- バックアップ/復元機能を追加し、全データ、競馬場別、WIN5のみ、回収率のみのJSONエクスポート/インポートに対応しました。
- AI自己進化ログを追加し、改善提案、採用/保留/却下、根拠レース、次回反映メモを管理できるようにしました。
- 保存ログ検索、最新保存ログ、競馬場別フォルダ、AI研究所、WIN5、万馬券DB、危険人気馬DBなどの運用フォルダを整理しました。
- GitHub Pages公開手順、Pages公開チェック画面、manifest、仮アイコンを整備し、ブラウザ運用版として公開できる構成にしました。

### Changed

- ブラウザ版・iPad対応版を正式なv1.0完成版として管理するため、READMEにv1.0セクションと完成版チェックリストを追加しました。
- リリース管理用に `RELEASE_NOTES.md`、`CHANGELOG.md`、`VERSION` を追加しました。

### Known limitations

- GitHub保存アダプターは準備実装で、GitHub Contents APIへの実保存/読込はPhase2予定です。
- 外部データ自動取得、複数端末同期、機械学習モデル本体、通知機能は今後の拡張対象です。
- v1.0は手入力・CSV取込・localStorage保存を中心にしたブラウザ運用版です。
