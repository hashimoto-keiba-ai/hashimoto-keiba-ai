# 橋本競馬AI v1.0 リリースノート

## v1.0 完成日

- **完成日**: 2026-06-02
- **バージョン**: 1.0.0
- **位置づけ**: ブラウザ運用版・iPad対応版を正式な「v1.0完成版」として管理する初回安定版

## 実装済み機能一覧

- **総合ダッシュボード**: 黒×金デザインのトップ画面で、今日のレース、AI指数、神穴、危険人気馬、WIN5、ROI、保存ログ、自己進化ログへ横断アクセスできます。
- **今日のレース一覧**: 開催場、発走時刻、距離、AI勝負度、注目馬、推奨操作、処理ステータスを一覧管理できます。
- **実データ入力フォーム**: レース条件、出走馬、人気、想定オッズ、脚質、AI指数、神穴指数、危険人気馬指数を手入力できます。
- **AI指数ランキング**: 登録した指数・人気・根拠をもとに、勝負候補をランキング形式で確認できます。
- **神穴ランキング**: 人気薄でも妙味がある馬を神穴指数・爆穴度・想定配当で整理できます。
- **危険人気馬判定**: 過剰人気、展開不利、馬場ミスマッチなどの観点で危険人気馬を管理できます。
- **三連単自動生成**: AI指数、神穴、危険人気馬を使い、本線・攻撃型・万馬券型の買い目生成に対応しています。
- **WIN5自動生成**: A固定、B本線、Cゾーン、D爆発候補を整理し、WIN5フォーメーションを生成できます。
- **回収率/ROIダッシュボード**: 投資金額、払戻金額、ROI、利益、的中率、券種別・競馬場別・月別集計を管理できます。
- **バックアップ/復元**: 全データ、競馬場別、WIN5のみ、回収率のみのJSONバックアップと復元に対応しています。
- **自己進化ログ**: 結果検証、改善提案、採用/保留/却下、次回反映メモを保存・確認できます。
- **保存ログ検索**: 競馬場、年月、レース名、騎手、WIN5対象などで保存ログを絞り込めます。
- **localStorage保存**: ブラウザ内保存により、PC/iPad/スマホ単体で運用データを保持できます。
- **GitHub Pages公開チェック**: Pages公開後のCSS/JavaScript/manifest/data JSON/localStorage/iPad表示を確認する導線を用意しています。

## 対応端末

- **PCブラウザ**: Chrome、Edge、Safariなどのモダンブラウザを想定
- **iPad**: Safariでの閲覧、ホーム画面追加によるアプリ風起動に対応
- **iPhone/スマホ**: 1カラム表示と下部固定ナビゲーションに対応
- **共通条件**: JavaScriptとlocalStorageが有効なブラウザで利用してください。

## GitHub Pages公開手順

1. GitHubで対象リポジトリを開きます。
2. **Settings** を開きます。
3. **Pages** を開きます。
4. **Build and deployment** で **Source: Deploy from a branch** を選びます。
5. **Branch: main** を選びます。
6. **Folder: /root** を選びます。
7. **Save** を押します。
8. 公開URLにアクセスし、トップ画面、CSS、JavaScript、manifest、data JSON、localStorage、iPad/スマホ表示を確認します。
9. ダッシュボード内の **Pages確認** または **Pages公開チェック** から、公開URLを入力してチェックを実行します。

## iPadホーム画面追加手順

1. iPadのSafariでGitHub Pages公開URLを開きます。
2. 共有ボタンを押します。
3. **ホーム画面に追加** を選びます。
4. 名前を「橋本競馬AI」にします。
5. **追加** を押します。
6. ホーム画面のアイコンから起動し、黒×金デザイン、CSV取込、AI指数、WIN5、ROI、バックアップ画面が開けるか確認します。

## 既知の制限

- GitHub保存アダプターは準備実装であり、GitHub Contents APIへの実PUT/GET、SHA更新、衝突解決はPhase2予定です。
- 出馬表、直前オッズ、馬場、天候、結果払戻などの外部データ自動取得は未接続です。v1.0では手入力・CSV取込中心の運用です。
- localStorage保存は端末・ブラウザ単位です。複数端末間の自動同期は未対応です。
- ブラウザのlocalStorage削除、プライベートブラウズ、端末初期化などで保存データが消える可能性があります。定期的にJSONバックアップしてください。
- AI判定は運用支援用であり、的中や利益を保証するものではありません。
- 現在のアイコンは仮SVGです。本番用アイコン画像は今後差し替え予定です。
